# Leoland Values and Themes v1

## Intended moral direction

Leoland should lean toward:

- good judgment over brute force
- leadership over domination
- long-term stewardship over short-term extraction
- fairness, tolerance, support, and responsibility
- learning from mistakes
- increasing sophistication by age/tier

## Themes already visible in the prototype

- trust matters
- cohesion matters
- civic responsibility matters
- environmental and educational choices matter
- legacy is not the same as raw power

## Things the project should avoid drifting into

- winning by being cruel
- cynical power fantasies
- military-maxxing as the dominant path
- shallow strongest-actor-wins loops

## Why this matters

These principles should shape later systems before mechanics harden, so the project remains recognizably Leo's world rather than becoming a generic strategy simulator.
