# Leoland Football Context Review v1

## Current posture

The football companion remains a separate sibling project.

Current server path:

- `/srv/leoland/leo-footy`

Current intake posture:

- present as a separate workspace
- not yet audited as a full code/product system
- not merged into the main Leoland repo

## What football can add to Leoland

Football is useful to Leoland when it helps express:

- regional identity
- civic pride
- belonging
- rivalry without hatred
- local mood
- media atmosphere
- the emotional texture of place

## What should become canon-supporting context

Useful football context later includes:

- club anchors per region
- rivalry tone
- shared rituals and festivals
- moments of pride or frustration that influence local mood
- headlines that help a region feel alive

## What should stay optional flavor for now

These should remain optional or out of core gameplay in this phase:

- match simulation in the main game loop
- detailed league tables driving national policy
- sports-management mechanics
- direct economy balancing through football systems

## Rule for L3

Football can enrich atmosphere and soft regional identity.
It should not take over the stewardship game.

## Honest note

The separate football repo/folder has not yet been deeply reviewed in this pass, so this is a relationship posture document rather than a full system-integration plan.
