# Leoland Turn Model v1

## Current shape

The first playable shell runs four turns.

Each turn:

1. shows the national state
2. shows the most stressed regions
3. offers three policy cards
4. asks the player to choose one card
5. asks the player to choose one response
6. applies national and regional consequences

## Starter deck posture

The starter deck contains twelve decisions across these themes:

- cost of living
- housing supply
- energy transition
- regional inequality
- trust in public institutions
- education / civic responsibility
- defence burden
- public services

## Regional layer

The current six-region layer is deliberately small and readable.

Each region carries:

- trust
- mood
- economic pressure
- public service pressure
- identity
- political temperament
- dominant issue

## What this is not yet

It is not yet:

- a province-level national simulator
- a party system model
- a coalition system
- a deep event chain engine

Those belong to the next deeper brief, not this one.
