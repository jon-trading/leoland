# Leoland First Playable Shell v1

## What L2 establishes

Leoland now has a small playable local shell rather than a hard-coded demo.

## Current playable loop

- start-of-game intro
- four-turn run
- one policy card choice per turn
- one response choice per policy card
- visible state summary after each turn
- end-of-run assessment

## Current state shown

- trust
- cohesion
- GDP
- education
- environment
- unrest
- legacy score

## Why this is enough for now

It makes Leoland feel like a real game loop without forcing:

- a public web app
- a database
- heavy architecture
- final balancing
