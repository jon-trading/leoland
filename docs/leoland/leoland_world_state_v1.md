# Leoland World State v1

## Purpose

Leoland now has a real world-state layer under the experience shell.

The point of this layer is to make the country feel alive without turning the game into a cold spreadsheet.

## Country state

The country state now tracks:

- turn
- season
- headline situation
- national public-good metrics
- treasury as a simple public-capacity abstraction
- active and completed scenario ids
- turn history

This gives the country memory and direction across a short session.

## Region state

Each region now carries:

- identity
- mood
- trust
- wellbeing
- pressure
- service strain
- belonging
- resilience
- current key issue
- headline and notes

This means a region is no longer just a name with a pressure label. It has social posture and civic texture.

## Institution state

Leoland models institutions before party theatre.

The first institution layer includes:

- schools
- healthcare
- housing
- transport
- local services
- emergency resilience
- community life
- civic information

Each institution carries:

- strength
- strain
- public trust
- fairness impact
- note

This keeps the game centered on how society actually functions.

## Issue tracks

Issue tracks are the main pressure channels that help activate scenarios and explain why the country feels tense or hopeful.

The first issue set includes:

- housing strain
- hospital queues
- flooding risk
- child wellbeing
- transport reliability
- cost-of-living pressure
- youth belonging
- energy affordability
- trust in public life

## Why this shape matters

This model supports a country that can:

- carry memory across turns
- spread pressure across regions
- show delayed consequences
- make fairness and resilience strategically meaningful

It is the first real substrate for a humane stewardship game rather than a handcrafted demo.
