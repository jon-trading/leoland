# Leoland Mobile Experience Shell v1

## Product stance

Leoland is now treated as mobile-first and desktop-friendly.

That means:

- vertical layout first
- short meaningful sessions
- touch-first navigation
- one-screen clarity where possible
- desktop play and development remain comfortable
- map and dashboard fused into one country-view experience

## First screen set

The new shell includes:

- Home / Country View
- Region View
- Decision View
- Reflection View

## Why this matters

The mobile shell gives the project a real emotional and aesthetic direction before the deeper simulation hardens.

It keeps the project from growing only as a terminal prototype while still preserving the existing terminal loop for fast iteration.
